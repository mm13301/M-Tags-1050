# M-Tags

**M-Tags** is a hospital equipment tracking system designed to help healthcare staff monitor, manage, and locate medical assets in real time. The goal of this project is to improve efficiency, reduce equipment loss, and support better patient care by making critical hospital equipment easier to track and manage.

---

## Overview

Hospitals rely on a large number of mobile assets such as IV pumps, wheelchairs, monitors, beds, and other medical devices. These items are often moved between rooms, departments, or storage areas, making them difficult to locate when needed.

**M-Tags** provides a centralized tracking platform that helps hospital staff:

- View equipment status in real time
- Track the location of hospital assets
- Manage multiple devices through a single dashboard
- Reduce time spent searching for missing equipment
- Improve workflow and asset utilization

---

## Features

- Real-time equipment tracking
- Equipment status monitoring
- Dashboard for viewing and managing tracked assets
- Ability to add and edit tracker modules
- Server-connected updates for reduced delay
- Expandable design for future hospital floor map integration
- User-friendly interface built for a hospital environment

---

## Purpose

The purpose of M-Tags is to solve a common problem in hospitals: misplaced or unavailable equipment. By improving visibility over where equipment is located and whether it is currently in use, M-Tags helps hospitals become more organized, efficient, and responsive.

This can lead to:

- Faster equipment retrieval
- Less unnecessary replacement spending
- Better resource allocation
- Improved support for hospital staff and patients

---

## How It Works

M-Tags connects physical tracking devices to a web-based platform. Each tracked asset can be monitored through the system, allowing users to view important information such as:

- Equipment ID
- Current status
- Assigned location
- Device details
- Tracker activity

The long-term vision includes mapping assets directly onto a hospital floor plan for even easier navigation and management.
